# The `rules` message
Will give all necessary information about the game.

```rules <rulename1> <value1> <rulename2> <value2>```

## Rules
- `randomizer`: either `uniform`, `7bag`, `genbag`, `unknown`
- `illegal-penalty`
- `max-illegal-count`
- `illegal-delay`
- `oncrash`: either `forfeit-game`, `forfeit-round`, `bugfix`
